Algorithm Course

  • Course 1

    Programming Basic 1
    (프로그래밍 기초)
    코드 작성 및 컴퓨팅 접근법 훈련.
    변수, 함수, 분기문 개념 훈련.

    Course 2

    Programming Basic 2
    (프로그래밍 기초)
    반복문 기초 및 데이터 반복 개념 훈련.
    TLE(Time Limit Exceed) 기초 대응.
    시간복잡도 개념의 이해.

    Course 3

    Programming Basic 3
    (프로그래밍 적응)
    자료구조를 제외한 기초 코딩 최종 훈련.
    산수가 아닌 수학적 기반 필요성 이해.

  • Course 4

    Array or List
    (배열 혹은 리스트)
    자료구조와의 첫만남, 배열과 리스트.
    자료가 구조화되어야 하는 이유의 이해.
    구조화된 자료를 통제하는 것이 결국
    알고리즘이다.

    Course 5

    String
    (문자열)
    "문자들의 배열"인 문자열에 대해
    학습한다. 문자는 글자고 글자이기에
    문화를 반영한다. 중요한 데이터이고
    배열과는 또다른 차이를 가지는 특수한
    자료인 문자열을 학습한다.

    Course 6

    Data Structure
    (자료구조)
    프로그래밍은 수많은 데이터를 다루고
    통제하는 것이 최종적인 목적이다.
    그래서, 프로그래밍에서 자료의 구조는
    매우 중요하다 할 수 있다. Stack,
    Queue, Tree, Set, Map, List,
    Deque 등 많은 자료구조를 학습한다.

  • Course 7

    Dynamic Programming Algorithm
    (동적계획법)
    subproblem 해결을 통한 문제들간의
    chain을 해결하는 방식이다.
    즉, 작은 문제들의 중첩으로 이루어진
    문제를 해결하는 방식을 학습한다.

    Course 8

    Graph Algorithm
    (그래프 알고리즘)
    Vertex(정점)와 Edge(간선)로
    이루어진 자료구조가 그래프 구조다.
    그래프 형태로 구조화된 자료를
    DFS(깊이우선), BFS(너비우선)
    탐색을 통한 문제해결 방식을 학습한다.

    Course 9

    Brute Force, Greedy Algorithm
    (완전탐색, 탐욕적 알고리즘)
    모든 경우의 수를 무식하게 전부 탐색
    한다고 해서 "짐승의 힘"이라는 브루트
    포스 알고리즘이고, 모든 순간에서 가장
    탐욕스럽게 최적의 선택만을 해 나간다고
    탐욕 알고리즘이라고 한다. 두 알고리즘
    을 통한 문제해결 방식을 학습한다.
